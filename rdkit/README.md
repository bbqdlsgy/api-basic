# RDKit Basic Usage for Cheminformatics
Practical RDKit methods frequently used in cheminformatics workflows,
including molecular descriptor calculation, visualization, and data export.

- Reading SMILES strings and generating RDKit Mol objects
- Calculating physicochemical descriptors (MW, LogP, TPSA, HBD/HBA)
- Visualizing molecular structures
- Exporting descriptor tables to Excel files
